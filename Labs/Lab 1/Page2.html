HTML lists allow web developers to group a set of related items in lists. There are three types of lists in HTML:

Unordered List:

An unordered list or bullet list starts with the <ul> tag. Each list item starts with the <li> tag. The list items are marked with bullets (small black circles) by default.

Example:
  • Item
  • Item
  • Item

Ordered List:

An ordered list or numbered list starts with the <ol> tag. Each list item starts with the <li> tag. The list items are marked with numbers by default.

Example:
  1. Item 1
  2. Item 2
  3. Item 3

Description List:

A description or definition list is a list of terms, with a description of each term. The <dl> tag defines the description list, the <dt> tag defines the term (name), and the <dd> tag describes each term.

Example:
Item 1
Description of Item 1
Item 2
Description of Item 2
