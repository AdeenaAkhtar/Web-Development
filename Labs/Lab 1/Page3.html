Tables can be used for more than just displaying a table of data. Tables can also be used as a formatting tool. The data in a table can be text or images. The <table> tag lets the browser know that a table follows.

Table Elements:

The attributes to the <table> element apply to the table itself not the data displayed in the table.

Table Row:
The <tr> tag defines table rows. The table header and table data elements are contained inside this element.
Table Header:
The <th> tag defines table columns, with each occurrence within a table row defining a new column. It is a cell element that indentifies the data to be displayed in the header of table.
Table Data:
The <td> tag defines table columns, with each occurrence within a table row defining a new column. It is a cell element that indentifies the data to be displayed in the rows of the table..

Table Attributes:

The attributes to the <table> element apply to the table itself not the data displayed in the table.

Align:
This attribute specifies the alignment of the table.

Width:
This attribute specifies the width of the table, not columns.

Border:
The border attribute specifies that the border of the table should be drawn.

Cellspacing:
This attribute specifies the spacing between cells

Cellpadding:
This attribute specifies padding within cells.
