HTML

A simple website built using only HTML to show all the it has to offer


HTML uses predefined tags that tell the browser how to display the content. Tags are nothing but some instructions that are enclosed in angle braces(i.e., < >). It is divided into three parts, i.e., opening tag, content(which will display on the browser), and closing tag, but some tags are special tags that do not contain closing tags like <br> tag.

Essential Tags:

The <!DOCTYPE html> declaration defines a document to be an HTML5 document
The <html> tag is the root element of an HTML page
The <head> tag contains meta information about the document
The <title> tag specifies a title for the document
The <body> tag contains the visible page content

Headings:

In HTML, there are six types of headings, i.e., h1, h2, h3, h4, h5, and h6. These headings are used to highlight the important topics.

<h1> defines the most important heading.


<h2> defines the second most important heading.


<h3> defines the third most important heading.


<h4> defines the fourth most important heading.


<h5> defines the fifth most important heading.

<h6> defines the least important heading.

Paragraphs And Breaks:

The <p> tag defines a paragraph.
The HTML <br> tag defines a line break.It is used to introduce a line break (a new line) without starting a new paragraph

Links:

HTML links are defined with the <a> tag. To create a link to Google.com we use: <a href="https://www.google.com" > Google </a>

Images:

HTML images are defined with the <img> tag. The source file (src), alternative text (alt), width, and height are provided as attributes. If the image is unavailable the alternative text is shown. . Eg. <img src="Assets\google.jpg" alt="google.com" width="100" height="100">:
